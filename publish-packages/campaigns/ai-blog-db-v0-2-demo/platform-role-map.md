# Platform Role Map

## note

Role: long-form Japanese article / search archive

Use note for the main Japanese argument. It should explain why AI-usable DB matters and preserve the idea as a searchable public article.

## Substack Post

Role: trust / email / subscriber asset

Use Substack Post as the relationship-building version. It should feel like a build log and deliver the core idea directly to subscribers.

## Substack Notes

Role: interaction / discovery

Use Notes for short questions, follow-up thoughts, and reader interaction inside Substack.

## X

Role: sharp hook / discovery

Use X for the strongest short claim and diagram-paired posts. Keep it concise and non-clickbait.

## Threads

Role: conversational discovery

Use Threads for a softer, conversational breakdown of the same idea.

## Instagram

Role: carousel / visual explanation

Use Instagram to turn the idea into a save-worthy visual sequence.

## YouTube Long-Form

Role: authority / deeper trust

Use YouTube long-form to explain the system, show examples, and build authority.

## Shorts

Role: short hook / reach

Use Shorts as a compact hook from the long-form script.

## Podcast

Role: reflective trust / audio relationship

Use Podcast for a slower commentary version that reveals the thinking behind the system.
