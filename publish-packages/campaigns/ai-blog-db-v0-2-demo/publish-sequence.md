# Publish Sequence

## Day 1: X / Threads Teaser

- Publish one X hook post.
- Publish one Threads conversational teaser.
- Use the X hook image if ready.
- CTA: ask whether readers are storing articles or reusable knowledge.

## Day 2: note Article

- Publish or finalize the note article.
- Use note hero image.
- CTA: point to the ongoing build log / Substack.

## Day 3: Substack Post

- Publish the build-log version.
- Add subscriber CTA.
- Include a short reader question.

## Day 4: Instagram Carousel

- Use carousel cover.
- Expand first 3-5 slides manually.
- CTA: save / follow / read full article.

## Day 5: YouTube Long-Form Outline Or Recording

- Record talking-head or hybrid version.
- Use screen inserts.
- TODO: thumbnail.

## Day 6: Shorts Cutdown

- Extract one sharp point from the YouTube script.
- Add captions manually.
- CTA: full video or Substack/note.

## Day 7: Podcast Commentary

- Record reflective commentary.
- Use podcast script as outline.
- Create show notes.

## Ongoing: Substack Notes Follow-Up

- Ask one reader-facing question.
- Share one behind-the-scenes note.
- Link back to Substack Post when natural.
