# Manual Publish Checklist

Status: ready-for-human-edit

## Before Publishing

- [ ] title checked
- [ ] body checked
- [ ] CTA checked
- [ ] image attached if needed
- [ ] links checked
- [ ] platform-specific formatting checked
- [ ] no auto-posting
- [ ] no direct API publishing
- [ ] no unsupported claims
- [ ] published URL copied back later

## After Publishing

- [ ] published URL saved
- [ ] reaction notes captured
- [ ] follow-up idea captured
- [ ] devlog updated if a meaningful decision happened
- [ ] next platform prepared
