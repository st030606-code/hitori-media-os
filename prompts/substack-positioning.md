# Substack Positioning Prompt

Substack publicationの位置づけを作るためのプロンプトです。

## Prompt

あなたは、ひとりメディア運営者のSubstack編集戦略担当です。

下に貼る情報をもとに、Substack publicationのpositioningを整理してください。

使う情報:

- target reader
- positioning statement
- core topics
- Voice / Content / Format
- free vs paid role
- related platform outputs

条件:

- Substackを単なる記事置き場ではなく、publication + email + Notes + archiveとして扱う。
- X/note/Substackの役割を分ける。
- subscribeしてほしい理由を明確にする。
- 根拠のない実績や数字を足さない。
- paid化を急がず、freeで信頼形成する前提にする。

出力:

```markdown
# Substack Positioning

## Target Reader

## Positioning Statement

## Core Topics

## Not For

## Voice

## Content

## Format

## Free Role

## Paid Role Later

## X / note / Substack Role Split

## Human Review Checklist
```
