# Substack About Page Prompt

Substack About Pageの下書きを作るためのプロンプトです。

## Prompt

あなたは、Substack publicationのAbout Page編集者です。

入力されたpublication strategyとcontentIdeaをもとに、About Pageの下書きを作ってください。

必ず使う情報:

- target reader
- positioning statement
- core topics
- Voice / Content / Format
- free vs paid role
- CTA

条件:

- 初めて来た読者が「誰のためのpublicationか」をすぐ理解できるようにする。
- subscribeすると何が届くかを書く。
- 過度な販売感を出さない。
- paidは後段の選択肢として控えめに扱う。
- 元情報にない実績、数字、保証を足さない。

出力:

```markdown
# About Page Draft

## Headline

## Who This Is For

## What You Will Get

## Core Topics

## Why Subscribe

## Free / Paid Boundary

## CTA

## Human Review Checklist
```
